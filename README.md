# Rick & Morty React App

En enkel React-applikasjon som henter data fra Rick & Morty API.  
Appen viser karakterer med bilde, informasjon og en egen detaljside for hver figur.  
Inneholder også søkefunksjon og navigasjon via React Router.

##  Teknologier
- React  
- Axios (API-henting)  
- React Router  
- Tailwind CSS

##  API
Data hentes fra: https://rickandmortyapi.com/
